Rewards System

This section provides an overview of in-game benefits and premium content available to players of the Ordo Draconis server. All features are obtained through gameplay activity — primarily through voting. This is not P2W — our goal is to maintain a fair gameplay environment without power advantages.

Premium

Premium is available in two variants:

  • Premium GAME — applies to one game account (and all characters on that account).
  • Premium WEB — applies to your entire web account (all linked game accounts and their characters).

How extensions work

  • GAME Premium + activation again = duration is extended (e.g., +30 days).
  • WEB Premium + activation again = duration stacks (e.g., +30 days).
  • Switching from GAME → WEB: if you have active GAME Premium and activate WEB Premium, WEB Premium replaces GAME and starts a new duration based on WEB Premium.

All activations are done using in-game currency earned through voting.

VIP 24h

VIP is obtained through voting:

  • 1 vote = 2 Vote Coins (VC)
  • Daily maximum = 8 VC
  • 5-day vote streak = +15 VC bonus
  • VIP 24h = 4 Vote Coins
  • 4 VC = 1 Dragon Coin (DC)
  • Vote Coins can be exchanged for Dragon Coins

VIP / Premium Benefits

  • .offline command (offline — selling / crafting / buffing via .sellbuffs)
  • .vip / .premium — display current VIP / Premium status
  • .exp_off / .exp_on — disable or enable EXP gain
  • .online — display current number of online players
  • additional QoL features will be added over time, always with a focus on non-P2W

Coming soon

We are preparing cosmetic features that do not affect character power (e.g., mounts, skins, etc.). The goal is to enhance the experience, not to provide PvP advantages.

Important information

  • All features and rewards are obtained through gameplay activity.
  • These features are designed to maintain game balance and follow a non-P2W approach.
  • Private trades between players are not supported or guaranteed (selling currency/items outside the system is prohibited).
  • The server may be temporarily unavailable due to maintenance, updates, or technical adjustments.
