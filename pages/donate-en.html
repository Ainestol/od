Donate

This page is dedicated to supporting the Ordo Draconis server. Donations are voluntary contributions for server operation and maintenance (hardware, hosting, development, security, fixes). This is not P2W — we do not want a system that gives anyone power advantages in-game.

Premium (supporting the server)

As a thank you for your contribution, you can obtain Premium in two variants:

  • Premium GAME — applies to one game account (and therefore all characters on that account).
  • Premium WEB — applies to your whole web account (all game accounts linked to it and all their characters).

How extensions work

  • GAME Premium + purchasing again = time is extended (e.g., +30 days).
  • WEB Premium + purchasing again = time stacks (e.g., +30 days).
  • Switching from GAME → WEB: if you have active GAME Premium and activate WEB Premium, WEB Premium overwrites GAME and starts a new period based on WEB Premium.

Benefits can also be obtained without paying.

VIP 24h (no payment required)

VIP does not have to be bought with money. It can also be earned through voting:

  • VIP 24h = 4 Vote Coins
  • or 1 Dragon Coin (value = 4 Vote Coins)
  • Vote Coins can be exchanged for Dragon Coins

VIP / Premium benefits

  • .offline command (offline trade / selling / crafting / buff shop via .sellbuffs)
  • more QoL benefits will be added over time, always with a non-P2W focus

Coming soon

We are preparing cosmetic features that do not affect character power (e.g., mounts, skins, etc.). The goal is a better experience — not a PvP advantage.

Important (fair and honest)

  • Donate = contribution to support the project, not a “purchase of goods” — therefore it does not create standard e-shop refund claims.
  • We do not handle or guarantee private trades between players (selling currency/items outside the system is forbidden).
  • This is a community project — we do our best for stability, but maintenance, downtime or fixes may occur.
