How to start

If you wish to enter the world of Lineage II Ordo Draconis, follow the steps below to begin your journey.

Step 1 – Download the game client

The download button for the game client is ready and waiting.

Game client

Step 2 – Game setup

After downloading, extract the files to a drive where you have enough free space.

Step 3 – Connecting to the server

Run the l2.exe file in the system folder.
For now, the game is available for testing only.
