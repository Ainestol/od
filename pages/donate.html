Ordo Shop

Tato část webu slouží k získání herních výhod a prémiového obsahu pro server Ordo Draconis. Zakoupené balíčky poskytují rozšířené možnosti ve hře a přispívají k dalšímu rozvoji serveru, jeho stabilitě a technickému zázemí. Nejde o P2W – cílem je zachovat férové herní prostředí bez výhod ovlivňujících sílu postavy.

Premium

Premium je dostupné ve dvou variantách:

  • Premium GAME – vztahuje se na jeden herní účet (a všechny postavy na tomto účtu).
  • Premium WEB – vztahuje se na celý webový účet (všechny herní účty i jejich postavy).

Jak funguje prodlužování

  • GAME Premium + nákup znovu = čas se prodlouží (např. +30 dní).
  • WEB Premium + nákup znovu = čas se sčítá (např. +30 dní).
  • Přechod z GAME → WEB: pokud máš aktivní GAME Premium a aktivuješ WEB Premium, WEB Premium přepíše GAME a začne běžet nová doba podle WEB Premium.

Výhody lze získat i bez nákupu

VIP 24h (bez nákupu)

VIP není nutné získávat nákupem. Je dostupné i prostřednictvím hlasování:

  • VIP 24h = 4 Vote Coiny
  • nebo 1 Dragon Coin (hodnota = 4 Vote Coiny)
  • Vote Coiny lze směňovat za Dragon Coiny

Výhody VIP / Premium

  • .offline příkaz (offline – prodej / výroba / buff .sellbuffs)
  • .vip / .premium – zobrazení aktuálního stavu VIP / Premium
  • .exp_off / .exp_on – vypnutí a zapnutí získávání EXP
  • .online – zobrazení aktuálního počtu online hráčů
  • další QoL benefity budou doplněny postupně, vždy s důrazem na non-P2W

Do budoucna

Připravujeme možnost pořízení kosmetických věcí, které nemění sílu postavy (např. mounti, skiny apod.). Cílem je lepší herní zážitek, ne výhoda v PvP.

Důležité informace

  • Zakoupené balíčky představují digitální obsah a služby v rámci hry.
  • Tyto výhody neovlivňují herní rovnováhu a jsou navrženy jako non-P2W.
  • Soukromé obchody mezi hráči nejsou podporovány ani garantovány (prodej mimo systém je zakázán).
  • Server může být dočasně nedostupný z důvodu údržby, aktualizací nebo technických zásahů.
