Reward systém

Tato sekce poskytuje přehled herních výhod a prémiového obsahu dostupného hráčům serveru Ordo Draconis. Všechny funkce lze získat aktivitou ve hře — především hlasováním. Nejde o P2W — naším cílem je udržet férové herní prostředí bez výhod v síle postavy.

Premium

Premium je dostupné ve dvou variantách:

  • Premium GAME — vztahuje se na jeden herní účet (a všechny postavy na tomto účtu).
  • Premium WEB — vztahuje se na celý webový účet (všechny propojené herní účty a jejich postavy).

Jak funguje prodlužování

  • GAME Premium + opětovná aktivace = doba se prodlouží (např. +30 dní).
  • WEB Premium + opětovná aktivace = doba se sčítá (např. +30 dní).
  • Přechod z GAME → WEB: pokud máš aktivní GAME Premium a aktivuješ WEB Premium, WEB Premium přepíše GAME a začne běžet nová doba podle WEB Premium.

Všechny aktivace probíhají pomocí herní měny získané hlasováním.

VIP 24h

VIP lze získat hlasováním:

  • 1 hlas = 2 Vote Coiny (VC)
  • Denní maximum = 8 VC
  • 5denní streak = +15 VC bonus
  • VIP 24h = 4 Vote Coiny
  • 4 VC = 1 Dragon Coin (DC)
  • Vote Coiny lze směnit za Dragon Coiny

Výhody VIP / Premium

  • .offline příkaz (offline — prodej / výroba / buffování přes .sellbuffs)
  • .vip / .premium — zobrazení aktuálního stavu VIP / Premium
  • .exp_off / .exp_on — vypnutí a zapnutí získávání EXP
  • .online — zobrazení aktuálního počtu online hráčů
  • další QoL funkce budou postupně přidávány, vždy s důrazem na non-P2W

Do budoucna

Připravujeme kosmetické prvky, které nemění sílu postavy (např. mounti, skiny apod.). Cílem je lepší herní zážitek, ne výhoda v PvP.

Důležité informace

  • Všechny funkce a odměny jsou získávány herní aktivitou.
  • Systém je navržen tak, aby zachoval herní balanc a non-P2W přístup.
  • Soukromé obchody mezi hráči nejsou podporovány ani garantovány (prodej měny/itemů mimo systém je zakázán).
  • Server může být dočasně nedostupný z důvodu údržby, aktualizací nebo technických úprav.
